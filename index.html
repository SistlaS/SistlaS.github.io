Publication

Offset Value Coding in Database Query Execution

ACM Transactions on Database Systems (TODS), 2025

My work introduces an extension of offset-value coding—a lightweight encoding technique originally used in external merge sort—to accelerate full database query execution. By preserving and reusing these codes across operators and order-preserving plans, we eliminate recomputation costs and unlock performance gains across sorting, joins, grouping, and star-join workloads. The technique integrates cleanly into existing query planners and significantly speeds up order-preserving pipelines.
